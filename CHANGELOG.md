# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release
- 42 MCP tools for Obsidian integration
- Vault management (list, info, stats)
- Note CRUD operations (read, write, delete, move)
- Frontmatter/property management
- Tag management (CRUD, rename across vault)
- Task management with Obsidian syntax support
- AppleScript integration for app control
- URI scheme support for native Obsidian actions
- Comprehensive test suite (138 tests)

## [0.1.0] - 2024-01-15

### Added
- Initial release with full feature set
- File system operations for vault access
- AppleScript interface for macOS app control
- Obsidian URI scheme handler
- 42 MCP tools covering all major Obsidian operations
- Property-based search functionality
- Tag CRUD and bulk operations
- Task management with due dates and priorities
- Comprehensive documentation
- Full test coverage with mocking
